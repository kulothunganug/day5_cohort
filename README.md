
# Online Course Marketplace

## Description
A platform where instructors upload courses and students purchase them.

## Features
- JWT authentication
- Instructor / Student roles
- Course creation and lessons
- Razorpay course purchase
- Real-time chat in live classes

## User Flow
Instructor: Login → Create Course → Upload Lessons  
Student: Browse Courses → Purchase → Learn

## Tech Stack
Frontend: HTML CSS JS  
Backend: Your choice 
Realtime: WebSockets  
Database: MongoDB/MYSQL
